Electrum voor Android

Deze pagina legt uit hoe Electrum te installeren op Android apparaten.

Wees bewust dat Electrum niet wordt gedistributeerd wordt als binair pakket, maar als python broncode; dit betekend dat de installatie lichtjes moeilijker is dan een app downloaden van de Google play store.

Het is mogelijk om alles te installeren via QR codes.

Als je moeilijkheden ondervindt kan he help vinden op deze pagina.

1. Download en installeer Google Scripting Layer voor Android

Je kan het hier vinden, of je kan volgende QR code scannen:

2. Download en installeer Python voor Android

Je kan dit hier doen, of je kan volgende QR code scannen:



Eens je de apk geinstalleerd hebt start je de Python voor Android app en klik je op "installeer".

3. Download het Electrum installeer scrypt

1. Start SL4A.
2. Klik op menu.
3. Klik op Add.
4. Klik op Scan Barcode.
5. Scan volgende QR-code:

Dit installeert het script genaamd e4a_install.py

4. Download en installeer Electrum

1. Klik op e4a_install.py: het download en installeert de folder "e4a-1.7.3"
2. Om Electrum op te starten ge je naar de folder "e4a-1.7.3" en klik je op 'e4a.py'

5. Maak een snelkoppeling op je beginscherm

1. Ga naar je beginscherm, klik op menu en selecteer "Aan beginscherm toevoegen".
2. In het menu dat verschijn kies je "Directe toegang", dan "Scripts".
3. Ga naar het sript ea4.py
