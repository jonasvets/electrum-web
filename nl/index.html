Electrum

Lightweight Bitcoin Client



Electrum is een makkelijk te gebruiken Bitcoin software. Het beschermt je tegen verlies van Bitcoins door een backupvergissing of computerprobleem, want je portefeuille kan gerecupereerd worden via een geheime zin die je op kan schrijven of onthouden. Er zijn geen wachttijden bij het opstarten van de software, omdat deze de Bitcoin blockchain niet hoeft te downloaden.



Kenmerken:

  • Werkt onmiddellijk: De software download de blockchain niet, maar gebruikt een server.
  • Zorgeloos: Uw portefeuille kan gerecupereerd worden via een geheime zin
  • Veilig: Je geheime zin of private keys worden niet naar de server gestuurd. Informatie die van de server wordt ontvangen wordt geverifiÃ«erd met SPV.
  • Geen pannes: Verschillende servers zijn beschikbaar en je kan direct overschakelen op een andere.
  • Beschikbaarheid: Je kan dezelfde portefeuille vanop verschillende computers gebruiken, ze synchroniseren automatisch.
  • Open: Je kan je private keys exporteren naar andere bitcoin software.
  • Getest: Electrum is open source en werd voor het eerst beschikbaar in november 2011.
